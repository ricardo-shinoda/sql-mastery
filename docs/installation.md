How to install Docker on a Macbook

2. Como instalar a versão leve no Mac (OrbStack)
O OrbStack é uma alternativa ao Docker Desktop que consome muito menos bateria e memória, além de iniciar em 2 segundos. Para um MacBook profissional, é a escolha ideal.

Passo a passo no Mac:

Acesse orbstack.dev.

Baixe o instalador para o chip do seu Mac (Intel ou Apple Silicon/M1/M2/M3).

Arraste para a pasta Applications e abra o OrbStack.

Ele vai pedir para configurar as ferramentas de linha de comando. Aceite tudo.

No terminal do Mac, digite docker ps. Se não der erro, o ambiente está pronto!
